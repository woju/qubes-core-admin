.. program:: qvm-remove

====================================
:program:`qvm-remove` -- Remove a VM
====================================

Synopsis
========
:command:`qvm-remove` [*options*] <*vm-name*>

Options
=======

.. option:: --help, -h

    Show this help message and exit

.. option:: --quiet, -q

    Be quiet

.. option:: --just-db

    Remove only from the Qubes Xen DB, do not remove any files

.. option:: --force-root

    Force to run, even with root privileges
