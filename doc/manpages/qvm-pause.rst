.. program:: qvm-pause

:program:`qvm-pause` -- pause a domain
======================================

Synopsis
--------

:command:`qvm-pause` [-h] [--verbose] [--quiet] *VMNAME*

Options
-------

.. option:: --help, -h

   Show the help message and exit.

.. option:: --verbose, -v

   Increase verbosity.

.. option:: --quiet, -q

   Decrease verbosity.
